# 一期验收测试清单

## 1. 测试边界

一期验收对象为 1 台工程验证样车。测试场景限定为低速、人行道或小区道路、500 米以内、不过马路。无监督自动返回、主动绕障、开放道路交通能力不作为一期验收项。

## 2. 验收用例

| 编号 | 测试项 | 步骤 | 通过标准 | 证据 |
| :--- | :--- | :--- | :--- | :--- |
| T01 | 推行低阻力 | 切换推行模式或断开驱动使能，人工推动样车 | 推行无明显电机拖拽，不影响人工移动 | 现场记录/视频 |
| T02 | Web 访问 | 手机或电脑连接车载热点，打开控制台 | 页面可访问，状态正常刷新 | 截图 |
| T03 | 前后图传 | 打开前后视频流 | 前后画面可实时查看，无持续黑屏 | 截图/录屏 |
| T04 | 遥控运动 | 在空旷区域下发前进、后退、左转、右转、刹车 | 车辆按指令低速运动，刹车有效 | 现场视频 |
| T05 | 急停 | 运动中触发急停 | 车辆立即停止，解除前不再响应运动指令 | 现场视频 |
| T06 | UWB 跟随 | 用户携带标签低速行走 | 样车保持安全距离跟随，不出现持续冲撞或失控 | 现场视频 |
| T07 | 目标丢失停车 | 遮挡或关闭 UWB 标签 | 样车在超时后停车并提示异常 | 日志/视频 |
| T08 | 防撞停车 | 在前方近距离放置障碍物 | 样车停车等待，障碍移除后可由人工恢复 | 现场视频 |
| T09 | 数据记录 | 启动记录并完成一段路线 | 生成含图像、UWB、里程计、IMU、GPS 的数据包 | 数据包清单 |
| T10 | 远程接管 | 跟随或异常状态下切回遥控模式 | Web 遥控可接管车辆并恢复安全状态 | 现场视频 |
| T11 | 雷达接口预留 | 检查安装位、供电和数据接口 | 预留接口符合约定规格 | 照片/接线说明 |
| T12 | 文档交付 | 检查交付资料 | 脱敏 BOM、接线图、调试说明、测试报告齐备 | 文档清单 |

## 3. 建议验收口径

- 单项测试允许在现场调参后复测。
- 跟随演示以稳定低速跟随为目标，不要求复杂人群中的高速穿行。
- 障碍处理以“停车等待 + 人工接管”为一期策略，不要求主动绕行。
- 数据记录验收以数据完整性和时间戳可对齐为准，不要求一期完成 VTR 自动返回。

## 4. 交付前自测清单

- 电池满电并完成固定，线束无裸露风险。
- 急停按钮可物理触发并被软件状态识别。
- Web 端在手机和电脑浏览器均可访问。
- 摄像头、UWB、IMU、GPS、编码器至少连续记录 10 分钟。
- 控制命令限速生效。
- 所有异常状态会写入事件日志。
