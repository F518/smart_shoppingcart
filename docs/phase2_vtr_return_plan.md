# 二期 VTR 自动返回预研说明

## 1. 二期目标

二期目标是在一期真实采集数据基础上，实现限定场景下的视觉教学回放（Visual Teach and Repeat, VTR）自动原路返回。该能力不等同于开放道路自动驾驶，验收应限定路线、速度、光照、障碍策略和人工接管条件。

## 2. 技术路线

- 去程：样车跟随用户时记录前后图像、编码器里程计、IMU、GPS/UWB 数据。
- 建图：将路线转换为图像序列与里程计位姿组成的拓扑路径。
- 回放：返程时用当前摄像头画面与去程记录画面做匹配，修正航向和横向误差。
- 兜底：GPS/北斗提供粗定位，编码器和 IMU 提供短时航迹推算。
- 安全：遇到障碍先停车等待，超时后 Web 端报警并由人工接管。

## 3. 分阶段验证

| 阶段 | 距离 | 验证目标 |
| :--- | :--- | :--- |
| P1 | 100 米 | 路线数据可复用，车辆能沿简单路径低速回放 |
| P2 | 300 米 | 处理轻微光照变化和里程计累计误差 |
| P3 | 500 米 | 完成目标场景内完整路线回放和异常停车 |

## 4. 验收边界建议

- 场景限定为同一路线、不过马路、低速、人行道/小区道路。
- 不承诺白天采集夜间返回，或极端强逆光、暴雨、大面积施工等情况。
- 不做主动绕障，一期和二期默认策略均为停车等待与远程接管。
- 自动返回前需要完成路线教学或去程数据采集。

## 5. 二期报价依据

二期属于算法集成和实车联调工作，建议单独报价：

- VTR 自动返回工程增强版：7.8-9.8 万起，具体取决于是否已包含一期硬件。
- 若仅在一期样车基础上追加二期算法：建议增量报价 3-5 万。
- 若要求源码、算法文档和技术交底：进入 12-18 万源码/买断版。

## 6. 参考开源路线

- QVPR teach-repeat：低算力单目视觉教学回放，适合做二期算法起点。
- hoverboard-firmware-hack-FOC：可作为无刷轮毂电机 FOC 控制参考。
- Raspberry Pi / Jetson Orin Nano：根据视觉算法负载选择车载主控平台。
