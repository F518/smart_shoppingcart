# 智能购物车一期工程验证方案

## 1. 项目定位

根据现有沟通，本项目的一期目标不是直接做泛化自动驾驶购物车，而是先交付一台可实地演示、可采集数据、可远程接管的工程验证样车。该样车用于验证“最后一公里室外跟随 + 路径数据采集 + 后续自动返回算法基础数据”的业务闭环。

一期建议采用“UWB 稳定跟随 + 前后视觉采集”的路线：UWB 负责低速跟随的稳定性，前后摄像头同步记录路线图像，为二期视觉教学回放（VTR）自动返回积累真实数据。

## 2. 一期交付范围

### 包含内容

- 1 台工程验证样车的底盘动力、电控、传感器与主控集成。
- 室外低速 UWB 跟随功能，支持目标丢失后停车。
- 前后广角摄像头图像采集与视频查看。
- Web 端图传与遥控页面，支持前进、后退、转向、刹车、急停和状态查看。
- 编码器、IMU、GPS/UWB、前后图像的时间戳数据记录。
- 基础近距离防撞停车逻辑。
- 激光雷达外挂模块的供电与安装接口预留。
- 调试说明、脱敏 BOM、接线说明、测试报告。

### 不包含内容

- 一期不承诺无监督自动原路返回作为验收项。
- 不包含激光雷达硬件采购、雷达建图算法和后处理平台。
- 不包含开放道路通行、红绿灯识别、主动绕障、复杂路径规划。
- 不包含量产结构设计、认证测试、防水防尘等级认证。

## 3. 技术路线

### 跟随与控制

样车采用 UWB 跟随模块获取用户相对距离与角度，主控根据目标位置计算线速度和角速度，再通过底层电机控制模块驱动两驱差速底盘。车辆在目标丢失、通信异常或防撞触发时进入停车保护状态。

### 视觉与数据采集

前后摄像头用于实时图传和路线数据采集。系统会记录图像帧、编码器里程计、IMU、GPS/UWB 等同步数据，形成二期 VTR 自动返回算法可复用的数据包。

### Web 图传遥控

车辆提供本地 Wi-Fi 访问入口。测试人员可通过手机或电脑浏览器查看视频、切换模式、遥控车辆、触发急停，并查看电量、连接状态、传感器状态和当前记录状态。

## 4. 开发周期

一期建议周期为 6-8 周。

| 阶段 | 时间 | 目标 |
| :--- | :--- | :--- |
| 方案定型与采购 | 第 1 周 | 确认购物车包络尺寸、关键器件、结构安装方式 |
| 底盘与电控集成 | 第 2-3 周 | 完成电机、驱动、电池、主控、急停与基础运动控制 |
| Web 与数据记录 | 第 3-5 周 | 打通图传、遥控、状态接口和同步数据记录 |
| UWB 跟随联调 | 第 5-6 周 | 完成室外低速跟随、目标丢失停车、防撞停车 |
| 场地测试与交付 | 第 7-8 周 | 完成验收测试、文档整理和样车交付 |

## 5. 报价口径

### 方案 A：一期工程验证版

- 报价区间：人民币 48,000-58,000 元。
- 包含：1 台工程验证样车、一期软件系统、现场/场地联调、交付文档。
- 适合目标：快速验证业务闭环，采集真实路线数据，为二期自动返回做准备。

### 方案 B：一期增强版，包含自动返回攻关

- 报价区间：人民币 78,000-98,000 元起。
- 周期：在一期基础上增加 8-12 周自动返回算法部署与测试。
- 适合目标：甲方要求首期即验证 VTR 自动原路返回，但需要接受限定场景验收。

### 方案 C：源码与技术买断版

- 报价区间：人民币 120,000-180,000 元。
- 包含：核心导航/VTR、传感器融合、底盘控制等底层源码交付，以及技术交底培训。
- 适合目标：甲方计划内部二次开发或后续量产。

## 6. 验收边界

一期验收应聚焦可验证工程能力：

- 车辆可低速跟随指定 UWB 标签。
- 车辆可通过 Web 页面遥控和急停。
- 前后视频可查看，路线数据可保存和回放检查。
- 近距离障碍触发停车。
- 车辆可在推行模式下保持低阻力。

自动返回、主动绕障、复杂路径规划和道路交通能力不作为一期验收项。
