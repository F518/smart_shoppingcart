# 内部成本与报价测算

> 本文件仅供内部决策使用，不建议直接发给甲方。

## 1. 一期成本模型

一期默认交付 1 台工程验证样车，目标是用 UWB 跟随降低演示风险，同时完成二期 VTR 所需的数据采集链路。

### 硬件成本估算

| 模块 | 估算成本 | 备注 |
| :--- | ---: | :--- |
| 主控与存储 | 600-1,500 | 树莓派 4B/5、同级 ARM 板或按库存选择 |
| UWB 跟随模块 | 300-1,800 | 国内模块或成套 PDOA/AOA 方案，按稳定性选型 |
| 前后摄像头 | 200-600 | 150 度广角，720P/1080P，含支架与线材 |
| 电机与驱动 | 800-2,000 | 两驱轮毂电机或改装 FOC 方案 |
| 电池与电源 | 500-1,200 | 24V 电池、BMS、降压、电量监测 |
| IMU/GPS/防撞传感 | 150-500 | 基础定位、姿态、近距离防撞 |
| 结构件与辅材 | 800-1,500 | 钣金、3D 打印、线束、急停、防护 |
| 测试损耗与备件 | 1,000-2,000 | 备件、返工、运输、耗材 |
| **硬件合计** | **5,000-8,000** | 购物车本体、激光雷达另计 |

### 人力成本估算

| 工作项 | 人天 | 说明 |
| :--- | ---: | :--- |
| 方案确认与结构测绘 | 3-5 | 包络面、安装方式、电源接口 |
| 底盘与电控集成 | 8-12 | 电机控制、急停、推行/遥控模式 |
| Web 图传遥控 | 5-8 | 视频流、控制页面、状态接口 |
| UWB 跟随联调 | 8-12 | 距离角度解析、控制策略、丢失停车 |
| 数据记录链路 | 5-8 | 图像、里程计、IMU、GPS/UWB 同步 |
| 场地测试与文档 | 8-12 | 验收测试、故障修复、交付资料 |
| **合计** | **37-57 人天** | 约 1.5-2 人月 |

按内部综合人力成本 15,000 元/人月估算，人力成本约 22,500-30,000 元。加上硬件、测试、差旅与风险缓冲，一期内部硬性成本建议按 35,000-45,000 元看待。

## 2. 报价建议

### 推荐对外报价

- 一期工程验证版：48,000-58,000 元。
- 内部目标成交价：不低于 52,000 元。
- 底线成交价：不低于 48,000 元，且必须删除现场长期驻场、自动返回、激光雷达建图等范围。

### 上调报价触发条件

出现以下任一条件，应上调报价或拆分二期合同：

- 甲方要求一期验收无监督自动返回。
- 甲方要求开放道路、过马路、红绿灯识别或主动绕障。
- 甲方要求源码买断或核心算法可二次开发。
- 甲方要求 2 台及以上样车同步交付。
- 甲方要求激光雷达建图算法、云端平台或 ETC 结账系统接入。
- 甲方要求明确量产结构、防水防尘、认证测试。

## 3. 分层报价口径

| 方案 | 对外报价 | 内部定位 |
| :--- | ---: | :--- |
| 一期工程验证版 | 4.8-5.8 万 | 主推，风险可控，自动返回不验收 |
| 自动返回增强版 | 7.8-9.8 万起 | 需要限定场景与额外周期 |
| 源码/技术买断版 | 12-18 万 | 高阶锚点，适合甲方自研与量产 |

## 4. 商务保护建议

- 合同中写清“一期为工程验证样车，不是量产商品”。
- 自动返回只写“接口预留、数据采集、二期预研”，不要写入一期验收。
- 核心算法源码默认不交付，若交付必须切换到买断报价。
- 对甲方版本只提供脱敏 BOM，供应商与真实采购价不外发。
- 激光雷达模块明确“硬件、供电、安装、数据接口”和“建图算法”分开报价。
